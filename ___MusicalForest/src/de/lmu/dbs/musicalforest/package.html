Musical Forest: Note detection and transcription system based on Randomized Decision Trees.

This is a command line tool that can extract polyphonic note onsets and note offsets (including pitch), train forests upon audio data etc.

Dependencies:
  • Apache commons IO Version 2.2 (for file operations)
  • Apache commons Codec Version 1.6 (for hashing tools in forest meta files)
  • JDom 2.0.2 (for XML parsing)
  • JOpt Simple 4.4 beta 2 (for command line parsing)
  • MidiReference 1.0 (some MIDI helpers)
  • Trove 3.0.3
