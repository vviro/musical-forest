Java Audio Spectrum Analysis:

Implementations of FFT and Constant Q Transform Algorithms.

Also included:
  • Some analyzer tools to run over the spectral data, i.e. a peak detector class and a (very) rudimental implementation of Anssi Klapuris iterative, polyphonic f0 detection algorithm.
  • Some MIDI helper classes to work with the same data types as the spectra generated by the package.
  • A Helper class to store the spectral (or any 2-dimensional) arrays to PNG image.
  • Usage examples that stores to PNG and/or binary files.
  • Utilities to scale the spectral data.


The Transforms are all based on the RasmusDSP native Java FFT implementation (CQT also bases on FFT). The core CQT is based on an implementation of Karl Helgason and P. J. Leonard, optimized by Thomas Weber.

Dependencies:
  • RasmusDSP audio library (native java FFT implementation)
  • JDom 2.0.2 (for XML parsing)
  • JOpt Simple 4.4 beta 2 (for command line parsing; in the example only)
